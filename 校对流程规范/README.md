### 基本流程

校对人员必需使用播放器以正常观看视频的方式，载入字幕后完整看完视频

若遇到需修改处，使用 Aegisub 自行修改字幕

校对人员需检查 错译、漏译、错字、中文语法错误 等文本问题，这包括 对话 和 屏幕字 等所有文本的问题；  
还需要检查 闪轴 问题，详见教程里 时轴 相关章节；  
发现的问题即使反馈给上下流，例如检查到大量闪轴，说明时轴人员技术有问题，需要打回给时轴人员重做，并反馈给项目管理人员，让其重新调整项目规划


### 工具选用

播放器 和 Aegisub 必须使用 libass 作为字幕 filter，详见教程里 字幕filter 相关章节

[MPC-HC](https://github.com/clsid2/mpc-hc/releases/)
和
MPV
都内置 libass 且基本兼容倍速

如果不需要倍速，新版 PotPlayer 也内置了 libass，或者给没有内置 libass 的播放器外挂 `xy subfilter with libass` 滤镜

Aegisub 必须使用新版的 arch1t3cht 分支，详见教程中 工具 相关章节


### 注意事项

* 必须使用指定工具
* 必须完整观看视频
