## 单项目人员选取规范

单项目指的是一个独立的可完整处理完一部指定作品的项目

### 人员组成原则

人员组成应保证以下目标可达成

`#` 后的数字为大致优先级，实际项目按当时情形分配

* 翻译可用性
    1. 保证翻译文本产出且语义准确 #1
    2. 提高翻译准确度 #6
* 时轴可用性
    1. 保证时轴产出并与文本合轴 #2
    2. 保证规范处理闪轴 #7
* 样式可用性
    1. 保证样式符合规范 #3
    2. 保证样式美观 #8
* 压制可用性
    1. 保证成品的内封字幕正常渲染 #4
    2. 保证成品的画质在播放时与源的差距肉眼不可见 #5
    3. 按需判断是否添加滤镜、压制内嵌等 #9


### 细节规范

1. 翻译 & 校对 中，必须有水平达标者能覆盖整部作品。  
   例如 翻译 由 A、B 两人按前后十分钟分段翻译，全篇由 C 校对，  
   则要么 C 水平达标、不强制 A、B 水平，  
   要么 A、B 水平皆达标、不强制 C 水平。

3. 闪轴处理为本组对打轴人员的水平的强制要求。  
   但实际情况按需处理，例如当使用官方字幕等现成字幕时，不存在打轴人员，  
   类似的情况不强制要求处理闪轴，以按需节约一个人员。  
   若需简单处理现成字幕，可用时轴后处理器处理时轴后由校对校一遍。
