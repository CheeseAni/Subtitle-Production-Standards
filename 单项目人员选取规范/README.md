## 单项目人员选取规范

单项目指的是一个独立的可完整处理完一部指定作品的项目

### 人员组成原则

人员组成应保证以下目标可达成

\# 后的数字为大致优先级，实际项目按当时情形分配

* 翻译可用性
    1. 先保证翻译文本产出 #1
    2. 次保证翻译文本准确度 #5
* 时轴可用性
    1. 先保证时轴产出并与文本合轴 #2
    2. 次保证规范处理闪轴 #6
* 样式可用性
    1. 先保证样式符合规范 #3
    2. 次保证样式美观 #7
* 压制可用性
    1. 先保证内封可用 #4
    2. 次按需判断是否添加滤镜、压制内嵌 #8


### 细节规范

1. 翻译 & 校对 中，必须有水平达标者能覆盖整部作品  
   例如 翻译 由A、B两人按前后十分钟分段翻译，全篇由C校对，则要么C水平达标、不强制A、B水平，要么A、B水平皆达标、不强制C水平
2. 闪轴处理为强制要求，实际情况下，当使用已有字幕，如官方字幕时，不存在打轴人员，这时才不强制要求处理闪轴，但同样应该用时间后处理器处理时轴后由校对校一遍
