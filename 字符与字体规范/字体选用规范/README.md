## 字体选用

### 对话字幕必须使用无衬线非艺术字
如 `思源黑体`

### 保证地区集对应字形正确
**简中和原文部分必须正确**  
例如 `思源黑体` 简中必须用 `思源黑体 CN`，若原文为日文，则必须用 `Source Han Sans JP`

繁中按需择字体

### 禁止使用系统中自带的字体
如 `微软雅黑`  
避免有傻子乱安装

### 尽量避免使用有兼容性问题的字体
如 `思源黑体 VF`  
可变字体可能会导致某些版本的 Windows 10 崩溃

### 尽量避免同一部番的字幕中增加字体
前面集用过的字体放置在一个目录下，  
每次需要找的时候先从这个目录找之前用过的字体，  
无必要则尽量复用字形类似的字体，  
防止合集中的字体包体积爆炸

不对字体完全相似做强制要求，只要别差距太大就行



## 字体相关标签使用


### 尽量避免 `对无Bold字重字体调用\b1` 等操作
如 `方正兰亭黑` 系列有 `中黑` `中粗黑` 等一众字重，应通过更改对应字体名来改变字体粗细，尽量避免使用 `\b1`

### 禁止使用 `\i1`
若需调整倾斜角度，应使用 `\fax`
